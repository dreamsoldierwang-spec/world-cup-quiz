# ⚽ 2026美加墨世界杯知识挑战

> 🎨 **TRAE AI 创造力大赛** · 生活娱乐赛道作品

一个手绘漫画风格的 **2026 FIFA 世界杯答题 H5 页面**，每局随机抽取 20 题，挑战你的足球知识储备！

## 🌐 在线体验

**👉 [点击开始挑战](https://dreamsoldierwang-spec.github.io/world-cup-quiz/)**

## ✨ 功能亮点

| 特性 | 说明 |
|------|------|
| 🎨 **手绘漫画风格** | 手写体字体、涂鸦边框、暖色调配色，充满童趣 |
| 🎬 **开场动画** | 足球 + 国旗飞入动画，仪式感满满 |

| 📊 **排行榜** | LocalStorage 本地排行榜，记录你的最佳成绩 |
| 📱 **响应式设计** | 完美适配手机、平板、PC 各种屏幕 |

## 🖼️ 预览

### 封面页
手绘漫画风格的封面，带有吉祥物和三国国旗装饰，点击「开始挑战」进入答题。

### 答题页
- 顶部进度条 + 实时得分
- SVG 圆环倒计时器
- 题目分类标签（主办城市、赛制改革、球星等）
- 答对/答错即时反馈 + 解析

### 成绩页
- Canvas 绘制的段位卡片（渐变背景 + 角色形象 + 分数进度条）
- 支持下载保存为图片
- 排行榜 TOP 10

## 🎯 计分规则

- 每题基础分 **5 分**
- 时间奖励：剩余时间 / 20 × **5 分**
- 总分换算为 **100 分制**：`round(( earned / 150 ) × 100)`

### 段位划分

| 段位 | 分数 | 称号 |
|------|------|------|
| 👑 | 90+ | 传奇球王 |
| ⭐ | 75-89 | 全明星 |
| ⚽ | 60-74 | 资深球迷 |
| 📣 | 40-59 | 业余爱好者 |
| 🌱 | 0-39 | 足球萌新 |

## 🛠️ 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript（零依赖）
- **SVG 动画**：圆环倒计时（stroke-dashoffset）
- **Canvas API**：动态绘制段位卡片
- **LocalStorage**：本地数据持久化
- **Google Fonts**：Patrick Hand / Comic Neue / Noto Sans SC
- **GitHub Pages**：静态托管
